# Assignment1 Automated Instance Management Using AWS Lambda and Boto3

## 1. EC2 Setup

Go to the AWS Management Console → EC2 → Instances → Launch Instance.

Launch two t2.micro instances (free-tier eligible).

In Tag section:

Instance 1:
Key: Action
Value: Auto-Stop

Instance 2:
Key: Action
Value: Auto-Start
